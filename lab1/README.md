### Опис проєкту

* Цей репозиторій містить виконання першої лабораторної роботи, яка складається з двох частин:
* 1\. \*\*Частина 1:\*\* Завантаження (urllib) та очищення даних індексу VHI (NOAA) по областях України, реалізація вибірок та базової статистики.
* 2\. \*\*Частина 2:\*\* Аналіз датасету споживання електроенергії (Individual Household Electric Power Consumption), фільтрація за складними умовами (timeit), нормалізація даних та підрахунок кореляції Пірсона/Спірмена.
