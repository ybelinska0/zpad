# Лабораторна робота №7: Computer Vision та багатопотоковість у C++

## Опис проєкту
Ця робота є розширенням програми для обробки відеопотоку. Основна мета — інтеграція нейронної мережі для детекції облич та реалізація асинхронної обробки даних для збереження високої продуктивності інтерфейсу (FPS).

## Виконані завдання
- [x] **Детекція облич:** Інтегровано модель ResNet-10 (SSD) через модуль `cv::dnn`.
- [x] **Багатопотоковість:** Важкі обчислення (інференс нейромережі) винесені в окремий потік за допомогою `std::thread`.
- [x] **Синхронізація:** Для безпечного обміну даними між потоками використано `std::mutex` та `std::lock_guard`.
- [x] **Ефекти:** Додано режими інверсії, розмиття, Canny та Glitch-ефект.

## Архітектура програми
Проєкт побудований за принципом розділення обов'язків:
1. **Main Thread (Головний потік):**
   - Відповідає за зчитування кадрів з камери.
   - Малює графічний інтерфейс та обробляє натискання клавіш.
   - Відображає відео в реальному часі без затримок.
2. **Worker Thread (FaceDetector):**
   - Працює у фоні.
   - Отримує копію кадру, перетворює його у `blob (300x300)` та запускає детекцію.
   - Оновлює координати обличчя асинхронно.



## Керування
- `6` — Увімкнути детекцію облич (DNN).
- `1` — Звичайний режим.
- `2` — Інверсія кольорів.
- `3` — Розмиття (Gaussian Blur).
- `4` — Детектор меж (Canny).
- `5` — Glitch-ефект (зсув RGB каналів).
- `q` — Вихід з програми.
- **Слайдер Brightness** — Регулювання яскравості кадру.
