# Лабораторна робота №5: Наука про дані: обмін результатами та початковий аналіз

Ця лабораторна робота присвячена створенню інтерактивного веб-додатку за допомогою бібліотеки Streamlit. Програма дозволяє аналізувати та візуалізувати вегетаційні індекси (VCI, TCI, VHI) для різних областей України. Дані автоматично завантажуються з сервісу NOAA, очищуються від зайвих тегів та обробляються для подальшого аналізу.

## Вимоги до системи
ОС: Windows, macOS або Linux
Інтерпретатор: Python 3.9 або вище

## Встановлення залежностей
Щоб запустити проєкт локально, встановіть необхідні бібліотеки з файлу requirements.txt за допомогою наступної команди:
pip install -r requirements.txt

## Запуск додатку
Запустіть локальний сервер Streamlit за допомогою цієї команди в терміналі:
streamlit run app.py

Після цього додаток автоматично відкриється у браузері, де можна взаємодіяти з графіками та фільтрами.
