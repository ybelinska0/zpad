# Лабораторна робота №5: Наука про дані (Streamlit)

Ця лабораторна робота присвячена створенню інтерактивного веб-додатку за допомогою бібліотеки Streamlit. Програма дозволяє аналізувати та візуалізувати вегетаційні індекси (VCI, TCI, VHI) для різних областей України, використовуючи дані з попередніх досліджень.

## Вимоги до системи
ОС: Windows, macOS або Linux
Інтерпретатор: Python 3.9 або вище

## Встановлення залежностей
Щоб запустити проєкт локально, встановіть необхідні бібліотеки з файлу `requirements.txt`:
```bash
pip install -r requirements.txt
