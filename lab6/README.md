# Лабораторна робота №6: C++ OpenCV

Програма для захоплення відео з веб-камери в реальному часі та застосування фільтрів за допомогою бібліотеки OpenCV. Має модульну об'єктно-орієнтовану архітектуру.

## Вимоги до системи
- **ОС:** Linux (рекомендовано Ubuntu/Debian)
- **Апаратне забезпечення:** Наявність веб-камери (пристрій `/dev/video0`)
- **Інструменти:** CMake, компілятор C++ (gcc/g++)

## Встановлення та запуск

1. **Встановлення залежностей:**
   Запустіть скрипт для встановлення CMake та OpenCV:
   ```bash
   ./preinstall.sh
2. **Встановлення залежностей:**
   Скомпілюйте програму за допомогою CMake:
   ```bash
   ./build.sh
3. **Запуск:**
   Відкрийте збудований додаток:
   ```bash
   ./run.sh
