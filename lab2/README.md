# Лабораторна робота №2: Аналіз даних VHI

Ця лабораторна робота присвячена завантаженню та аналізу індексу здоров'я рослинності (VHI) для різних областей України за допомогою Pandas.

## Вимоги до системи та середовища
- **ОС:** Windows, macOS або Linux
- **Інтерпретатор:** Python 3.9 або вище
- **Середовище:** Jupyter Notebook
