# Лабораторна робота №3: Візуалізація даних

Ця лабораторна робота присвячена основам візуалізації даних за допомогою бібліотек Matplotlib та Seaborn на прикладі класичного датасету "Titanic". 

## Вимоги до системи та середовища
- **ОС:** Windows, macOS або Linux
- **Інтерпретатор:** Python 3.9 або вище
- **Середовище:** Jupyter Notebook
